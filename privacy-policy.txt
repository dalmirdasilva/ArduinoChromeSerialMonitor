Privacy Policy for Arduino Serial Monitor
Last Updated: 2025-03-16

Thank you for using Arduino Serial Monitor (the "Product"). 
This Privacy Policy explains how we handle user data in connection with the Product.

Arduino Serial Monitor does NOT collect, store, or transmit any personal or user data. This includes, but is not limited to:

Usage data (e.g., how you interact with the tool).

Any data transmitted through the Arduino Serial Monitor.

Since Arduino Serial Monitor does NOT collect any user data, there is no data to use, process, or analyze.

3. Data Sharing
Arduino Serial Monitor does NOT share any user data with third parties, as no data is collected in the first place.

Arduino Serial Monitor does NOT integrate with any third-party services that collect or process user data.
